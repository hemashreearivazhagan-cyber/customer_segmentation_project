# 🧹 Customer Segmentation - Data Cleaning & Preprocessing

## 📌 Project Overview

This project focuses on cleaning and preprocessing customer data to prepare it for customer segmentation and further data analysis. The dataset is processed using Python and Pandas by handling duplicates, checking missing values, standardizing data, and exporting a clean dataset for analysis.

---

## 🎯 Objectives

- Load customer dataset
- Check dataset information
- Identify missing values
- Remove duplicate records
- Standardize categorical data
- Rename columns for consistency
- Generate summary statistics
- Export cleaned dataset

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- Visual Studio Code

## 📊 Dataset Information

The dataset contains **200 customer records** with the following attributes:

| Column | Description |
|---------|-------------|
| CustomerID | Unique customer identifier |
| Gender | Gender of the customer |
| Age | Customer age |
| Annual Income (k$) | Annual income in thousand dollars |
| Spending Score | Customer spending score (1–100) |

---

## 🧹 Data Cleaning Steps

The following preprocessing operations were performed:

- Loaded the dataset using Pandas
- Displayed dataset information
- Checked for missing values
- Removed duplicate records
- Standardized text values
- Renamed column names
- Verified data types
- Generated descriptive statistics
- Saved the cleaned dataset
